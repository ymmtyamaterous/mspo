# 全体概要仕様書

## プロジェクト名

**MinorSports（マイナースポーツ図鑑）**

## 概要

世界中のマイナースポーツ（知名度の低い競技）のルール・歴史・特徴を学べる情報サイト。ユーザーはスポーツを閲覧・検索でき、登録ユーザーは新しいスポーツ情報を投稿・編集できる。

---

## 目的・ゴール

| 目的 | 詳細 |
|------|------|
| 情報提供 | 世界各地のマイナースポーツをわかりやすく紹介する |
| 発見体験 | 知らなかったスポーツとの偶然の出会いを提供する |
| コミュニティ形成 | ユーザーが情報を投稿・拡充できる仕組みを提供する |

---

## ターゲットユーザー

- スポーツに興味がある一般ユーザー（10代〜40代）
- マイナースポーツの普及・啓発に関わる関係者
- 珍しいスポーツをランダムに探したいユーザー

---

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | TypeScript / React / TanStack Router / Vite |
| バックエンド | TypeScript / Hono / oRPC / Bun |
| データベース | SQLite / Drizzle ORM |
| 認証 | better-auth |
| モノレポ管理 | Turborepo |
| コード品質 | Biome / Lefthook |
| UI コンポーネント | shadcn/ui |

---

## システム構成

```
apps/
  web/       ← フロントエンド（React + TanStack Router）
  server/    ← バックエンド API（Hono + oRPC）
packages/
  api/       ← oRPC ルーター定義
  auth/      ← better-auth 設定
  db/        ← Drizzle スキーマ・マイグレーション
  env/       ← 環境変数バリデーション
  ui/        ← 共通 UI コンポーネント
  config/    ← 共通 tsconfig
```

## 主要機能一覧

### 一般ユーザー（未ログイン）

| # | 機能 | 説明 |
|---|------|------|
| 1 | スポーツ一覧閲覧 | 登録されたスポーツのカード一覧を閲覧できる |
| 2 | スポーツ詳細閲覧 | 競技名・ルール・歴史・発祥地などの詳細を確認できる |
| 3 | キーワード検索 | 競技名・キーワード・国名でスポーツを検索できる |
| 4 | カテゴリ絞り込み | 格闘系・水上・アウトドア等のカテゴリで絞り込める |
| 5 | ランダム表示 | ランダムに1件のスポーツを表示する |
| 6 | 今週のピックアップ閲覧 | 管理者が設定した注目スポーツを閲覧できる |
| 7 | ランキング閲覧 | 閲覧数・人気度順でスポーツを一覧できる |

### 登録ユーザー（ログイン済み）

| # | 機能 | 説明 |
|---|------|------|
| 8 | スポーツ投稿 | 新しいスポーツ情報を投稿できる（管理者承認後公開） |
| 9 | スポーツ編集 | 自分が投稿したスポーツ情報を編集できる |
| 10 | お気に入り登録 | スポーツをお気に入りに追加・削除できる |
| 11 | アカウント管理 | プロフィール情報（名前・アイコン）を変更できる |

### 管理者

| # | 機能 | 説明 |
|---|------|------|
| 12 | コンテンツ承認 | 投稿されたスポーツ情報を審査・公開/却下できる |
| 13 | ピックアップ設定 | 今週の特集スポーツを設定できる |
| 14 | カテゴリ管理 | カテゴリの追加・編集・削除ができる |

---

## 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンシブ対応 | PC・タブレット・スマートフォンに対応 |
| ページ表示速度 | 初期表示 3秒以内（目標） |
| 認証セキュリティ | better-auth によるセッション管理、HTTPS 前提 |
| アクセシビリティ | セマンティック HTML・alt テキスト対応 |

## デザイン方針

`docs/user/sample.html` のデザインを基準とする。

- **カラーパレット**
  - 背景: `#f7f3ed`（クリーム紙）
  - テキスト: `#1a1410`（ダークインク）
  - アクセント: `#e8450a`（レッドオレンジ）
  - サブアクセント: `#1a6e3c`（ダークグリーン）
  - ミュート: `#8a7d6e`
- **フォント**: Bebas Neue（見出し）/ Noto Sans JP（本文）
- **トーン**: 雑誌・図鑑風のエディトリアルデザイン

## 画面構成（ルーティング）

| パス | 画面 | 認証要否 |
|------|------|---------|
| `/` | トップページ（ヒーロー・カテゴリ・特集） | 不要 |
| `/sports` | スポーツ一覧ページ | 不要 |
| `/sports/:id` | スポーツ詳細ページ | 不要 |
| `/sports/random` | ランダム表示（詳細へリダイレクト） | 不要 |
| `/categories` | カテゴリ一覧 | 不要 |
| `/ranking` | ランキングページ | 不要 |
| `/submit` | スポーツ投稿フォーム | 必要 |
| `/my/favorites` | お気に入り一覧 | 必要 |
| `/my/submissions` | 自分の投稿一覧 | 必要 |
| `/login` | ログイン | 不要 |
| `/signup` | サインアップ | 不要 |
| `/admin` | 管理者ダッシュボード | 管理者のみ |
